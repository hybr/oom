# Entity Architecture

This directory contains architecture documentation for entities and their relationships.

## Contents

- **`ENTITY_CREATION_RULES.md`** - Complete rules and standards for creating entity metadata
- **`relationships/`** - Domain-specific entity relationship documentation

## Purpose

This directory defines:
- Entity naming conventions
- Attribute standards and data types
- Validation rules and constraints
- Entity relationship patterns

## When to Use

Consult these documents when:
- Designing new entities
- Adding attributes to existing entities
- Creating relationships between entities
- Understanding entity architecture
